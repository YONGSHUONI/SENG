Database guide

Access via JDBC
To use JDBC you need to load the mysql driver
Class.forName("org.gjt.mm.mysql.Driver").newInstance();
and then open a connection,
String DB="jdbc:mysql://127.0.0.1:3306/ + "Database Name"
e.g.
String DB="jdbc:mysql://127.0.0.1/mydatabase";
Conn = DriverManager.getConnection(DB);

Access via command line:
% command -h143.167.9.237 -u "your username"
